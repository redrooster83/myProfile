ChatGPT

  • Command Line Prompt
    • In December 2022 I heard about ChatGPT and immediately registered an account. After enjoying many conversations I suddenly wasn't able to access the site.
    I started looking into the possiblity of interfacing with ChatGPT through Python commands or general text commands, that would be able to carry out intelligent functions such as slicing data or creating visualisations.
    Instead of finding a solution for that, I found a couple of forums that helped me put together Python built prompted interactive chat with ChatGPT.
  • Data Visualisations using ChatGPT
    • My team uses mstrio for MicroStrategy on a regular basis for various functions, and the module comes built with various functions to interact with MicroStrategy's REST API.
    I wondered if I could combine ChatGPT with mstrio, but so far I was only able to create a matplot visualisation which I still find as an interesting.

Python

  • pyautogui Automatically Move Mouse Cursor
    • I was asked by a client to create a Python based program that would automatically open a file in a program and perform certain keystrokes.
    I started researching about pyautogui and during the course, I realised I could quickly put together a python script to automatically move my mouse cursor.
    As silly as it sounds, it helps me during workouts at home to stop my screensaver kicking in, so that I can skip through songs on my Spotify.
  • Python cgi scripts
    • I had an idea one night if it was possible to utilize our existing installation of MicroStrategy to host a Python application, instead of going through the trouble of installing a Python web framework.
    I came across a website indicating that it was possible to use cgi scripts hosted in the Tomcat Webapps folder, and I managed to build a cgi script webpage that uses Python to connect to the MicroStrategy API to refresh cubes through a button displayed on a webpage.
  • Python REST API
    • After spending some time with MicroStrategy to implment a POC using their Hypercard feature, I got very curious on how I could use the built in API feature.
    Through a lot of trial and error, and many late nights I was able to get a firm understanding of how the process worked, and realised the power that REST API's can give to providing different ways to interact with data.

HTML

  • Xampp
    • When I began studying web development almost 12 years ago, it was quite a hard concept to fully understand how web servers, php servers and databases all fit together.
    I probably should have shopped around for a better course, but now after attempting to pick up the technology again due to updates in technology that MicroStratgy has offered, it's suprisingly easy to quickly get things up and running using Xampp.
    MicroStrategy's Web and Library feature can both be deployed on Tomcat, and with their Library feature being able to work with their own REST API, it's a great opportunity to start exploring those skills using an easy to understand application.
  • PHP
    • I was working on a word document that I found quite frusting due to the multiple formatting options that had been set. It made the document quite difficult to fill out, and I found that I spent most of my time removing lines in the document which had been placed there intended to be printed out.
    I thought that I could try to solve this problem using an HTML form tag for each question that could write my answers into a database based on my profile, and once I've filled out every question, I would be able to build an analysis of each of my answers.
    I hadn't looked at PHP in a very long time, so this was a great opportunity to re-learn this skill after 12 years, and it would give me the opportunity to build a relational database. I have really enjoyed learning PHP and have gained some extra skills on the relational database development.
